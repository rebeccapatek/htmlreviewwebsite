HTML

Provides the structure to the content appearing on a website

HTML stands for HyperText Markup Language

Now what is a markup language you may be asking yourself? A markup language is a compute language that defines the structure and presentation of raw text. HTML allows a computer to interpret raw text that is wrapped in HTML elements Hypertext is text displayed on a computer or device that provides access to other text through links (also known as hyper links)
