HTML

Provides the structure to the content appearing on a website

HTML stands for HyperText Markup Language

Now what is a markup language you may be asking yourself? A markup language is a computer language that defines the structure and presentation of raw text. HTML allows a computer to interpret raw text that is wrapped in HTML elements Hypertext is text displayed on a computer or device that provides access to other text through links (also known as hyper links)

Let's Talk about elements

Html is composed of elements. These elements structure webpage and define its content.
EX: a paragraph element. A paragraph element is made up of
  1. Opening tag (<p>)
  2. Content (the text that you are displaying on the webpage. Ex: "Hello World!"
  3. Closing tag (<p>)

So basically the HTML element is a unit of content in an HTML document formed by the HTML tags and the text or media that it contains.

The body element is a key HTML element used to build a webpage. Only content inside the opening and closing body tags can be displayed on the screen. (although some browsers do accommodate for code outside the body element. So usually most of the code on your screen is nested in the body element.

Let's talk about Structure

When an element is contained inside of another element, it is considered the child of that element. So for example if you have a paragraph tag with content inside of a body element, then you would say that the paragraph is considered the "child" of the "parent" body element. You could also say that it is nested in the parent element.

For readability, it is common practice to add two spaces of indentation for the child of a parent element.

There can be multiple levels of nesting (grandchildren, great-grandchildren, etc.) The relationship between elements and their ancestors is known as hierarchy.

One of the reasons that understanding the HTML hierarchy is important, is because the child elements can inherit behavior and styling from their parent element.

Here's an example:

   <body>
     <h1></h1>
     <p></p>
   </body>

Both the header and the paragraph would be considered children of the body, while the h1 and p would also be considered siblings.

Let's talk about the div tag

A div is another important element in HTML. Div stands for division and they are defined as a container that divides the page into Section. These are very useful for grouping elements in your HTML together. Divs don't inherently have a visual representation, but when we want to apply custom styles using CSS to our elements, divs allow us to group the HTML elements and apply the same styles for all HTML elements inside.

Attributes are a way to expand an element's tag.

  • These are content added to the opening tag of an element.
  • These can be used to provide information or to change styling.

Attributes have two parts

  1. The name of the attribute. (ex: id)
  2. The value of the attribute. (ex: "intro")

So here's what a div would look like with this attribute:

   <div> id="intro">
     <h1>Introduction</h1>
   </div>

Displaying text in HTML

Paragraph or span are two common tags

  • paragraphs contain a block of plain text.
  • span contains short pieces of text or other HTML (is an inline element). They are used to separate small pieces of content that are on the same line as other content.

BLOCK-LINE ELEMENTS have a line break before and after them. div and p examples. BLOCK LINE ELEMENTS will take up the entire width of the its containing element. By default, <div> or <p> will push sibling elements to a new line or section of the page. One difference with div is that it will not skip a line

INLINE ELEMENTS are inside a sentence or paragraph. Span tags are an example. INLINE ELEMENTS are a more conservative use of space. They only take up as much space as they need and will not force surrounding content to a new line.

span tags are great when you want to make one word bold, italic, a different color.

There are a few HTML tags that style the text.

  • <em> This is short for emphasis. It will italicize the text within this tag
  • <strong> will make the text bold.
  • <br> is the line break element in HTML. It is different from the other tags in that it is only composed of a starting tag. It will send the following text to the next line on the screen. It will not leave an empty line, but if you would like to add an empty line, just put the <br> twice.
  • <ul> is and unordered list tag used to create a bulleted list. The <ul> element should not hold raw text, rather the nested list element enclosed in <li></li>
  • <ol> is an ordered list and is used to create a numbered list. <ol> would have to have the list elements nested as <li></li>

Images

<em> is the image tag that allows you to add an image to a webpage. The image tag is a self-closing tag so that the end of the image tags has a forward slash.

The image tag has a required attribute src (stands for source) and must be set to the images source or the location of the image. In some cases, the value of src must be a URL.URL stands for Universal Resource Locator.

It is important to use the alt attribute when putting an image on your website . This should be a description of the image. Here are the purposes of the alt attribute:

  1. If an image fails to load on the webpage, then the user can mouse over the area originally intended for the image and read a brief description of what was meant to be there.
  2. Visually impaired users often browse the web by using screen reading software.When you The alt attribute allows the screen reading software to read the image's description to the visually impaired user.Might be an ADA law as well?
  3. The alt attribute plays a role in SEO (stands for Search Engine Optimization) because search engines cannot see and image. Having descriptive alt attributes can help improve the ranking of your site.

Video

Video tags require the src attribute as well. Unlike image tags they require an opening and closing tag.

Here's and example of a video tag

<video src="myVideo.mp4" width="320" height="240" controls> Video not supported </video>

Video not supported will only be displayed if the video doesn't load

The width and height attributes are used to set the size of the video displayed in the browser.

The controls attribute instructs the browser to included basic controls: pause, play, skip.

Alt tags are not supported in video.
